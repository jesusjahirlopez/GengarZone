Acerca de nosotros

Esta página fue diseñada como el proyecto para la materia de Laboratorio de Progrmación Web, el equipo de desarrollo de este proyecto está conformado por estudiantes de la FIME, el objetivo de esta página es impulsar a las personas a expresar sus opiniones acerca de los múltiples temas que se pueden charlar, siempre y cuando se mantenga el respeto entre todos los usuarios, de lo contrario se tomarían las medidas necesarias.
